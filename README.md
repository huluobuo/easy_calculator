# 简易计算器

## 功能特性

✅ 基本四则运算（加减乘除）
✅ 科学计算功能（三角函数、幂运算、根号）
✅ 角度/弧度切换
✅ 实时算式显示
✅ 键盘快捷键支持
✅ 错误处理与提示

## 使用说明

1. 克隆仓库

```bash
git clone https://github.com/huluobuo/easy_calculator.git
```

2. 运行程序

```bash
python main.py
```

## 操作指南

- **基本运算**：直接点击数字和运算符
- **科学计算**：使用√(平方根)、∛(立方根)、^（幂运算）等按钮
- **角度切换**：顶部按钮切换"度/弧度"模式
